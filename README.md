# ContactHub

ContactHub is a Django web application for managing contacts in a clean and structured way.
It allows users to add, view, update, and delete contacts through a simple web interface.

## Features
- Create, update, and delete contacts
- Clean CRUD-based Django architecture
- Server-rendered UI
- Simple and intuitive layout

## Tech Stack
- Django
- HTML
- CSS

## Run Locally
```bash
git clone https://github.com/Malcom-Yingwani/contacthub.git
cd contacthub
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
